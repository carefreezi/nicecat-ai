### 介绍
奈斯AI程序目前版本V1.3.3
授权方式：三个顶级域名+两次更换AI绘图专业设计
包含功能点：
1.对话
2.绘画
3.思维导图
4.小应用
5.知识库
6.PDF解析
7.Ai写作
（Ai模型+星火大模型+文心一言）


### 系统亮点
~~~
1.无刷新加载
2.可对接多个key进行切换
3.切换指定模型运行
4.数据库保留对话记录
5.技术栈：前端nuxt3-后端laravel9开发
~~~


### 运行环境

```
Nignx/Apache
PHP 8.1
MySQL 5.7-8.0
Redis
```
